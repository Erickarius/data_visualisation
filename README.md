# Data Visualization Project

Welcome to the Data Visualization Project! This project focuses on data visualization and aims to showcase various techniques, tools, and libraries used for analyzing and presenting data in a visually appealing and informative manner.

## Project Description

This project provides examples and demonstrations on how to create impressive data visualizations using different tools and technologies. It also includes sample data for visualization and instructions on how to run and utilize the examples.

The project is based on the book "Python Crash Course 2nd Edition" by Eric Matthes.

## Project Structure

The project consists of the following directories and files:

- `mapping_global_data_sets/`: Contains examples of visualizing global datasets using maps.
- `plotting_simple_line_graph/`: Contains examples of creating simple line graphs.
- `random_walk/`: Contains an example of visualizing random walks.
- `rolling_dice/`: Contains an example of visualizing dice rolling results.
- `the_csv_file_format/`: Contains an example of working with the CSV file format.
- `work_with_api/`: Contains an example of working with APIs.

## Requirements

To use the data visualization examples in this project, you need to have the following tools and libraries installed:

- Python 3.x
- Matplotlib library
- Requests library
- Plotly library

You can install them using pip package manager by running:

`pip install matplotlib requests plotly`


Additional dependencies for each example are described within their respective directories.

## Instructions

1. Clone this project to your local machine.
2. Navigate to the directory of the chosen visualization example.
3. Follow the instructions provided in the documentation to run the example on your system.
